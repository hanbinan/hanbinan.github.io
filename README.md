# [《熨斗的呼吸》](https://hanbinan.github.io/)

深夜十一点，陈静的熨斗开始呼吸。

那是一台老式蒸汽熨斗，不锈钢底板磨成了镜子。丈夫和孩子睡下后，她把熨衣板支在阳台。月光从晾晒的衬衫缝隙漏进来，在水泥地上切出模糊的格纹。

嗤——蒸汽升腾的瞬间，某种东西在胸腔里松动。

她熨丈夫的衬衫。领口要转着圈熨，那是他动脉跳动的地方。去年体检发现颈动脉斑块后，他说话开始小心翼翼，像在冰面上行走。她熨得格外仔细，把每一道褶皱都烫成平原。

接着是儿子的校服。膝盖处总有洗不掉的青草渍。十四岁男孩的世界很大，大得装不下母亲的询问。“妈你别管”成了他最新的口头禅。她对着污渍呵气，蒸汽模糊了眼镜片。

最后才是自己的真丝衬衣。淡紫色，三年前买的，只穿过两次。一次是同学会，一次是独自看电影。真丝娇气，不能直接熨。她垫上白棉布，隔着布料感受温度传递。这很像她的生活——总隔着一层什么。

嗤——又一蓬蒸汽。

她忽然想起母亲也有这样一台熨斗。童年记忆里，母亲总在深夜熨衣服。那时不懂，为什么非要等到夜深人静。现在她明白了：只有夜晚的水汽能溶解白天的坚硬。

熨斗划过衣襟，像犁过冻土。底下有什么东西在苏醒。

上周三，她在超市遇见初恋。对方推着购物车，车里坐着双胞胎女儿。他们寒暄了天气、油价和孩子的补习班。全程五分钟，谁也没提那个暴雨的午后——他站在她家楼下，衬衫湿透贴着胸膛，说我要去南方了。她躲在窗帘后，咬着手背没哭出声。

当时她穿着这件淡紫色衬衣。

嗤——蒸汽轻吻衣料。

熨到第三件时，一滴水落在熨衣板上。不是蒸汽凝结的。她摸了摸脸颊，干的。那滴水从哪来？她抬头看晾衣绳，丈夫的深蓝领带正缓缓摆动，像深海里的某种生物。

熨斗越来越重。不是实际的重量，是二十三年婚姻、十八年育儿的密度全沉淀在手腕。她忽然渴望一次失手，让滚烫的底板直接贴上皮肤，留下一个疼痛的印记，证明某些东西真实存在过。

但她只是调低了温度。

全部熨完时，凌晨一点。她把衣服一件件挂回衣柜，像把不同的自己收回抽屉。丈夫的衬衫挂左边，儿子的校服挂中间，她的真丝衬衣蜷在最右侧，淡紫色在阴影里几乎看不见。

关灯前，她最后摸了摸熨斗。底板微温，像某个熟睡者的额头。

嗤——它在梦中又呼出一小口白气。
